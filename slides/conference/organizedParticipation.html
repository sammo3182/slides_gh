研究问题

纪律要求下的有组织社会化有用吗？

能改变青年的什么？怎么改变的？

  • 毛泽东1937年为抗日军政大学题写的校训。
  • 习近平2023年2月7日，在学习贯彻党的二十大精神研讨班开班式上提出，推进中国式现代化是一个系统工程，需要正确处理好顶层设计与实践探索、战略与策略、守正与创新、效率与公平、活力与秩序、自立自强与对外开放等一系列重大关系。

青年政治社会化与中国特色制度

  • 有组织社会化：以党支部、团支部为基本单元、持续周期性、要求成员参加
    • 社会主义社会化形式 → 中国特色：覆盖最广、制度化最高
  • 大学是中国青年政治社会化的最后一站，也可能是最重要的一站

习近平：人才培养体系贯通其中的是思想政治工作体系。 这一制度性社会化过程的实证研究，是理解中国高等教育独特运作逻辑的关键。

理论张力悬而未决

支持视角 (李丁 2017; 张良驯 and 郭元凯 2020)

  • 制度化嵌入 → 降低社会化成本
  • 周期性仪式 → 价值内化
  • 社会资本 → 政治效能感

怀疑视角 (Scott 1985; Huang 2015)

  • 强制性 → 工具化抵触
  • 高强度动员 → 逆反情绪
  • 缺乏弹性 → 政治冷漠

实证缺口长期存在

  1. 规范层面讨论多 → 实证检验少
    • 仅有研究基本基于欧美选举型、志愿型参与
  2. 焦点集中思政课程 → 参与式社会化被忽视

两类视角都有零散实证证据：基层选举、生态文明、英模宣传等场景， 但缺乏能同时容纳情感、行为、价值多层次结果的系统框架。

分析框架：情感 → 行为 → 价值

  • 政治心理学：态度形成的三层递进结构 (Seligman 2002)
    • 情感：认同的心理起点
    • 行为：认同的外在表现
    • 价值：认同的深层支撑
  • 行为认知机制：自我知觉 + 认知失调
    • 反复参与 → 自我归因 → 态度重塑 (Bem 1972; Festinger 1957)

这一框架将”行为先于态度”的微观机制引入分析， 为统一解释正向与负向证据分歧提供了视角。

现实预期

  • 有组织社会化 → 情感(获得感/幸福感/安全感) ↑ + 行为(制度化渠道偏好) ↑ + 路经(制度/政策/文化/路径认同) ↑
  • 相对于自愿参与，有组织社会化效应更全面、稳健
  • 体制外家庭学生 → 边际提升 ↑（补偿性社会化）

调查数据

  • 院系分层抽样，10% 在校生
  • 长期效应

访谈数据

  • 全国 17 所高校，51 名学生

  • 东西南北中地理覆盖

  • 解释变量：党团活动参与频次（vs 志愿服务频次）
  • 结果变量：情感（3）+ 行为（2）+ 价值（4）= 九个指标
  • 模型：累积链接 + 多层线性 + 交互项

有组织社会化改变了什么

有组织社会化

  • 获得感/幸福感/安全感 ↑
  • 制度化渠道 ↑ & 非制度化渠道 ↓
  • 四类认同 ↑

自愿性社会化

  • 获得感/幸福感 ↑/安全感↓
  • 制度化渠道 ↑ & 非制度化渠道
  • 四类认同 ↑

党组织凭借资源配置、价值引导与行为规范的系统性优势， 显著提升个体主观情感状态。志愿服务效应集中于情感连接维度。

有组织社会化的”双向效应”是关键发现： 不仅提升对正式渠道的信任，还主动规避非制度化行为。 这一双向性是自愿性社会化无法达成的。

价值认同维度上两条路径均产生显著效应， 但有组织社会化的优势在于覆盖更为全面的认同维度。 四个子维度与”四个自信”在结构上高度对应。

调节效应：补偿性整合

  • 体制外家庭学生的边际收益显著更大
  • 缩小社会出身带来的政治认同差距

体制内家庭学生进入大学时已具备较高政治认同起点，边际提升空间有限； 体制外家庭学生从制度化教育中获得更大补偿。 这印证了 Neundorf 等关于补偿性社会化的理论预期。

从效应到机制

  • （潜在）问题
    • 内部有效性？→ 是不是基于混淆变量的“假有效”？为什么会有效？
    • 外部有效性？→ 是否只是 T 大学的特殊现象
  • 解法
    • 混淆变量（如“学工经历”、“党员身份”）控制排除“假有效”
    • 质性数据回答“为什么
  • 答案
    • 访谈分析揭示四重机制的递进协同
    • 约束 → 濡化 → 示范 + 竞争
      • 制度准入 → 情感联结 → 行为转化 → 持续卷入

访谈在985到专科五类院校间普遍验证了 T 大学发现的外部有效性。 四重机制并非并列，而是逐层递进、相互嵌套， 共同实现”由外到内”的认同转化。

四重机制的类型学

机制 核心动力 渠道 支持类型
约束 制度强制 纵向 特定性
濡化 情感浸润 纵向 弥散性
示范 榜样感召 横向 弥散性
竞争 利益激励 横向 特定性
  • 约束 = 基础性条件 → 为其他机制提供制度土壤
  • 濡化 & 示范 = 弥散性支持 → 长期情感依附
  • 竞争 = 工具性卷入 → 持续动力

约束机制并非以强制压抑自主，而是为濡化、示范、竞争提供稳定的接触场域。 弥散性支持 vs 特定性支持的区分（Easton 1965）在此找到微观印证。

总结与启示

  1. 有组织社会化稳定塑造青年政治认同 → 三维全覆盖
  2. 纪律性 > 自愿性 → 推翻”强制无效”的经典预期
  3. 补偿性整合 → 弥合社会出身差异
  4. 四重机制协同 → “外在动员—内在信念”的转化路径

理论贡献：参与塑造效力取决于制度化深度与持续性，而非自愿/强制二分
实践启示：保留约束节奏 + 嵌入仪式濡化 + 重视身边榜样 + 谨慎竞争激励
学科视角：以政治心理学勾连马列研究、思想政治教育、人才培养等多领域

[正确处理]规范有序和激发活力的关系……筑牢高校意识形态安全防线。深化教育综合改革，以改革添动力、增活力，让广大教师潜心育人、大胆探索，让广大学生朝气蓬勃、追逐梦想。

“严”与”爱”、“管”与”育”在思政工作中的辩证统一。 体制外家庭学生边际收益较高，可在参与机会供给上适当倾斜。

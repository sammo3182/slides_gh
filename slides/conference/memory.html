Past → Today: A Three-Step Mechanism

Priming → Evaluation → Spillover

\(H_1\): Activating people’s SARS memory changes their view of the current authorities, mostly at the central level. \(H_{2.0} (Analogy)\): The activation of the SARS memory increases people’s approval of the government’s anti-COVID performance. \(H_{2.1} (Anchor)\): The activation of the SARS memory decreases people’s approval of the government’s anti-COVID performance.

heuristic processing information theory, individuals tend to perceive what they learned first as “normal,” even if it represents a cognitive bias, and use it as a benchmark to evaluate subsequent events.

Spillover

\(H_{3.0}\): People with activated SARS memory trust the government more.
\(H_{3.1}\): People with activated SARS memory trust the government less.

1.7 Large random sample properties

  • Convergence: \(p\lim_{n \to \infty}X_n = a, a\in R.\)
  • When a sequence of random variables stabilizes to a certain probabilistic behavior as n ‚Üí ‚àû , the sequence is said to show stochastic convergence.
  • Two Views of Convergence
    1. In probability: Values in the sequence eventually take a constant value (i.e. the limiting distribution is a point mass).
    2. In distribution: Values in the sequence continue to vary, but the variation eventually comes to follow an unchanging distribution (i.e. the limiting distribution is a well characterized distribution)
  • Consistency:
    \(p\lim_{n \to \infty}\hat{\theta}_n = \theta.\)

An estimator qn is consistent if the sequence q1‚Ä¶qn converges in probability to the true parameter value Œ∏ as sample size n grows to infinity

2 Confidence Interval

2.1 Learning the population from a sample

  • X: a random sample from a probability distribution with parameter Œ∏.

  • Confidence interval (CI) for the parameter Œ∏, with confidence level Œ≥: An interval determined by random variables u(X) and v(X) with the property: \(Pr\{u(X)<\theta <v(X)\}\ =\ \gamma = 1 - \alpha, \quad \forall\theta.\)

  • Human language: In a repeatedly sampling, the percentage of the samples that could contain Œ∏

2.2 Calculated CI (Two-tailed)1

  • \(\bar X \pm Z_{\alpha/2}SE\)
    • Œ±: 1 - Confident level;
    • Z-score: \(Z = \frac{X - \mu}{\sigma}\) Learned
  • Proportion: \(\pi = P \pm Z_{\alpha/2}\sqrt{\frac{P(1 - P)}{n}}\)

s is the sample standard distribution, \(SE(\bar{X}) = \sqrt{\frac{s^2}{n}}\)

2.5 Estimating Method

  • For mean
    • œÉ known, \(\mu = \bar X \pm Z_{\alpha/2}\frac{\sigma}{\sqrt n}\)
    • œÉ unknown
      • N ‚â• 100ish, then \(\bar X \pm Z_{\alpha/2}\frac{s}{\sqrt n}\);
      • N < 100ish, then \(\bar X \pm t_{\alpha/2}\frac{s}{\sqrt n}.\)
  • For proportion
    • œÄ known, \(\Pi = P \pm Z_{\alpha/2}\sqrt{\frac{\pi(1 - \pi)}{n}}\);
    • œÄ unknown, \(\Pi = P \pm t_{\alpha/2}\sqrt{\frac{\pi(1 - \pi)}{n}}\).
  • Degree of Freedom: Student‚Äôs T critical points are relative to the d.f.
    • For CI: n - 1; for regression: n - k - 1

3.3 Comparing Samples

Assuming X1 and X2 are i.i.d,

  • œÉ is known, \(\mu_1 - \mu_2 = (\bar X_1 - \bar X_2) \pm Z_{\alpha/2}\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}\).
  • œÉ is unknown, \(\mu_1 - \mu_2 = (\bar X_1 - \bar X_2) \pm t_{\alpha/2}\sqrt{\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}}\)
    • \((\bar X_1 - \bar X_2) \pm t_{\alpha/2}S_p\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}\) when the population of the samples are identical, i.e., \(\sigma_1 = \sigma_2\), where \(S_p^2 = \frac{\sum(X_1 - \bar X_1)^2 + \sum(X_2 - \bar X_2)^2}{(n_1 - 1) + (n_2 - 1)}\), d.f.: \((n_1 - 1) + (n_2 - 1)\), a.k.a., the ‚ÄúDifference in Means‚Äù.
  • Aggregated (propotional) data: \(\Pi_1 - \Pi_2 \pm Z_{\alpha/2}\sqrt{\frac{P_1(1 - P_1)}{n_1} + \frac{P_2(1 - P_2)}{n_2}}\)

3.7 Difference in Difference in Means

Independent sample

\[\mu_1 - \mu_2 = (\bar X_1 - \bar X_2) \pm t_{\alpha/2}\sqrt{\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}}\]

Matched samples

\(D = X_1 - X_2\), then \(\Delta = \bar D \pm t_{\alpha/2}\frac{S_D}{\sqrt{n}}\), where \(S_D = \frac{\sum(D - \bar D)^2}{n - 1}\).

Matched proportions (Aggregate data)

\(\hat{P}_D = \frac{n_{10} - n_{01}}{n}\), then \(\Delta = \hat{P}_D \pm Z_{\alpha/2} \cdot \sqrt{\frac{n_{10} + n_{01}}{n^2}},\) where \(n\) = Number of matched pairs, \(n_{10}\) = Number of pairs where the first observation is a success and the second is not, \(n_{01}\) = Number of pairs where the first observation is a failure and the second is a success.

This formula uses the discordant pairs (\(n_{10}\) and \(n_{01}\)) to calculate the confidence interval for the difference in proportions in matched samples.

Why using a different method for matched samples? Not IID for individuals, but might for the difference

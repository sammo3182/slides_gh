本章重点

  1. 矩在描述数据分布中的作用

本章研究问题：民众的平等观念如何分布，是更看重机会平等还是结果平等？

1.1 描述统计计算

  • 数据分析的核心：寻找规律
  • 描述性统计是寻找规律的起点
  • 用得对、用得好：为后续分析奠定基础，为读者建立”第一印象”
  • 本章核心变量：equalIncentive（平等观念）
  • 1 = 完全倾向结果平等，10 = 完全倾向机会平等
  • 中间数代表对两端不同程度的偏向
  • 最直接的描述方式：看各档位人数
  • 两种常用方法：
    • R 自带 table()：直接计算频数，向量形式
    • dplyr::count()：表格形式，更规整、支持多变量联合计数
  • table() 返回向量，每个值出现的次数
  • count() 返回 tibble，含 equalIncentive 与 n 两列
  • 多变量计数：第一行结果即”安道尔的某位男性选择了 equalIncentive=2”
  • 计数直观但难以提炼变量”特点”
  • 统计学家用”矩”描绘变量特征
  • 四阶矩对应四个直观概念：
    • 一阶矩：平均数
    • 二阶中心矩：方差（与之相关：标准差 = 方差的平方根）
    • 三阶标准化中心矩：偏度
    • 四阶标准化中心矩：峰度
  • 把每个值的人数画成柱状图，连成一条平滑曲线 → 密度图
  • 四个矩对应这座”山”的四个特征：
    • 平均值 = 山峰位置
    • 方差 = 山的宽窄
    • 偏度 = 山坡陡峭程度
    • 峰度 = 山顶尖锐程度
  • 知道这些特征，就能大致勾勒出山的样态

矩的悖论

  • 实际社科数据往往不呈完全钟形分布
  • 比如平等观念真实分布更像”山坡”
  • 平均值常出现在”半山腰”而非山峰
  • 矩的实际描述含义在偏态数据上会打折扣
  • 更稳妥的方式：将原始数据可视化（后文会讲）
  • 计算具体矩的函数：
    • mean()：平均数
    • var()：方差
    • sd()：标准差
  • summary() 一次返回六组：最小值、Q1、中位数、平均数、Q3、最大值

统计量与缺失值

  • 单独计算矩时通常加 na.rm = TRUE
  • R 默认不会忽略任何值——包括缺失值
  • 缺失值含义是”未知”，不是 0、不是”没有”
  • 含 NA 的向量直接运算会返回 NA
  • na.rm = TRUE 显式告诉 R”先移除缺失值再算”
  • dplyr::summarise() 一次输出多种统计量
  • group_by() 让运算在各组中分别进行
  • 二者搭配：群组层级分析的核心组合
  • 例：考察各国总体教育状况与年龄差异
  • 教育水平用样本均值估计
  • 年龄差异用标准差估计
  • group_by() 给数据加”分组”元数据
  • summarise() 对每组分别执行计算，消耗一层分组
  • ungroup() 显式移除剩余分组标记，恢复普通表格
  • 结果：把分析单元从”个人”拔高到”国家”，便于宏观比较

练习1：按收入水平的平等观念分布

不同收入水平（incomeLevel）的群体，他们的平等观念（equalIncentive）在分布上是否存在差异？

请用上面学到的函数回答这个问题。

  • 参考”按国家分组”的写法
  • 把分组变量改为 incomeLevel
  • 用 filter() 去掉 incomeLevel 的缺失值
  • 用 summarise() 计算各组的样本量、均值与标准差
library(dplyr)
data("wvs7", package = "drhur")

wvs7 |>
  filter(!is.na(incomeLevel)) |>
  group_by(incomeLevel) |>
  summarise(
    n = n(),
    equal_mean = mean(equalIncentive, na.rm = TRUE),
    equal_sd = sd(equalIncentive, na.rm = TRUE)
  ) |>
  ungroup()

# 解析：
# 1. filter 去掉 incomeLevel 缺失观测
# 2. group_by 按收入档位分组
# 3. summarise 算每组的样本量、平等观念均值与标准差
# 4. ungroup 恢复普通表格
# 5. 观察 equal_mean：高收入组若更高，说明该群体更倾向"机会平等"

1.2 描述统计可视化

  • 传统描述性统计以表格呈现
  • 但表格对非钟形数据的特征表现有限
  • 越来越多学者倾向使用直方图、密度图、小提琴图等
  • 表格仍是最基本的展示方式
  • 经验丰富者能从描述性表中判断数据质量
  • 但要丰富、全面才能达到这种效果
  • 推荐工具：modelsummary（Vincent Arel-Bundock 出品）
  • datasummary_skim()：一行代码生成带直方图的概览表
  • 既有传统统计量（均值、中位数、标准差）
  • 又附带嵌入式直方图，直观看分布
  • 按变量类型分别生成：
  • datasummary() 支持完全定制
  • 公式语法：<变量(行)> ~ <统计量(列)>

练习2：定制描述统计表

请使用 datasummary() 函数为 wvs7 创建一个描述性统计表，要求：

  1. 分析 age、education、equalIncentive 三个变量
  2. 用 datasummary 的公式语法控制内容
  3. 只展示四种统计量：N（非缺失观测数）、Mean（平均值）、Median（中位数）、SD（标准差）
  • 公式语法：<变量1> + <变量2> + ... ~ <统计量1> + <统计量2> + ...
  • 三个变量在 ~ 左边
  • 四个统计量在 ~ 右边
  • data = wvs7 作为参数
library(modelsummary)
data("wvs7", package = "drhur")

# 公式语法：变量(行) ~ 统计量(列)
datasummary(age + education + equalIncentive ~ N + Mean + Median + SD, data = wvs7)

# 解析：
# - 对比三变量的 N，可看 equalIncentive 是否比 age、education 缺失更多
# - 若 age 的 Mean 与 Median 接近 → 分布对称
# - 若 Mean 明显大于 Median → 分布右偏

1.3 常用统计图形

  • 直方图：最常见的一维分布展示
  • 把数据分成若干区间，矩形高度表示频数
  • 适用于连续变量
  • 离散变量更适合条形图（geom_bar）
  • 密度图：直方图的平滑版本
  • 通过核密度估计（kernel density estimation）：每个观测值上放一条”小山丘”曲线，相加后平均化
  • 优点：清晰展示分布形状，尤其在识别多峰分布时

连续变量才适合密度图

  • 密度图假设变量在连续轴上取值
  • 定序、分类变量两相邻类别间不存在连续空间
  • 核密度估计会在”空白区域”制造虚假趋势
  • 严格来说，经典直方图也是为连续变量设计的
  • 离散变量画”直方图”本质是条形图
  • 箱线图（box plot, 盒须图）：直接基于描述统计量展示分布
  • 箱体：从 Q1 到 Q3
  • 中间横线：中位数
  • 须线：从箱体向两侧延伸到 ±1.5 IQR 内的极值
  • 须线外的点：潜在离群点
  • 从图中可读出：
    • 中心趋势：中位数约 40 岁
    • 离散程度：IQR 约 28-53 岁
    • 偏态：上须线明显比下须线长 → 右偏
    • 异常值：无（所有数据都在须线内）
  • 箱线图的局限：完全基于统计量，可能严重误导
  • 三种截然不同的分布可能画出完全相同的箱线图
  • 这是因为统计量恰好一致而原始分布完全不同
  • 小提琴图（violin plot）：箱线图的更好替代
  • 垂直方向：分布的整体形状
  • 水平方向：核密度估计，宽窄代表密度
  • 与箱线图比：完整揭示分布轮廓（长尾、偏态、多峰、离群点聚集）
  • 与密度图比：在垂直轴上呈现，多组比较更紧凑
  • 例：比较不同平等观念群体的年龄分布
  • 用 ntile() 把 10 档简化为 3 组（结果平等更重要 / 二者都重要 / 机会平等更重要）
  • 关键观察：
    • 三组分布形态大致相似（集中在 25-50 岁）
    • “结果平等更重要”组在 25、37.5、50 岁附近各有隆起
    • “机会平等更重要”组分布更均匀，高龄尾部延伸更长
  • 这些细节是箱线图无法呈现的
  • 图形函数命名规律：以 geom_ 开头
  • geom = “geometry”（几何）的缩写
  • 后跟图形类型英文：geom_histogram、geom_density、geom_boxplot、geom_violin

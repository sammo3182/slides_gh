1.1 基本运算

  • R中的运算符与数学一致：+、-、*、/
  • 在Console中直接书写算式即可运算

空格的使用

  • 空格不影响运算，但能提高可读性
  • Tidyverse style guide推荐使用空格和空行
  • 可用styler、air等包统一格式
  • 幂运算^、取余%%、对数log()/log10()、指数exp()
  • 常用缩写命令：sqrt()（平方根）、abs()（绝对值）
  • 三角函数：sin()、cos()、tan()、acos()、asin()、atan()
  • pi是R预存的圆周率常量
  • 其他预存常量：letters、LETTERS、month.name、month.abb

科学计数法

  • e连接系数与指数，表示10的幂
  • 例：200 = 2e2；0.002 = 2e-3
  • 布尔代数：TRUE/FALSE（可简写T/F）
  • 逻辑运算：&（与）、|（或）、!（非）
  • 空集对象：NULL
  • 不等式判断：==、!=、>、<、>=、<=
  • 布尔运算次序放在最后：先算不等式，再算逻辑
  • 单定值参与布尔运算时，会进行自相等判断

等于的使用

  • ==表达等于，=留作赋值符
  • 初学者常误用=代替==，多加练习即可

1.2 向量

  • 向量（vector）：一串有方向的数
  • 创建方式：c()、:（连续整数）、seq()
  • 向量索引：[]提取子集
  • a[2]提取第2个元素，a[c(3,4)]提取多个元素

赋值符<-

  • <-是R最常用的赋值符，箭头表示”存进”
  • 快捷键：Windows/Linux Alt + -，Mac option + -
  • R几乎是唯一使用<-的流行语言，源自APL键盘布局
  • 向量加减法：对应元素相加减
  • 标量与向量运算：标量作用于每个元素
  • 元素积（Hadamard积）：*，对应位置相乘
  • 内积（点乘）：%*%，结果为标量
  • 张量积（外积）：%o%或outer()，结果为矩阵

向量的“非常规”运算

  • R中a / b为逐项除法，数学上向量无除法
  • 回收原则（recycling rule）：短向量自动重复匹配长向量
  • 可用c()创建字符型向量，需加双引号

现学现练

练习1：向量回收原则

根据a和e的赋值，解释并推算下面算式的结果。

  • a长度为5，e长度为2
  • R会将e循环扩展为c(1, 2, 1, 2, 1)
  • 然后逐元素进行运算
a <- 11:15
e <- c(1, 2)

# e 被循环为 c(1, 2, 1, 2, 1)
# a - e: c(10, 10, 12, 12, 14)
a - e

# a / e: c(11, 6, 13, 7, 15)
a / e

练习2：推演干支纪年系统

利用向量操作推演六十甲子。规则：阳干配阳支，阴干配阴支。

  • 用seq()和%%区分奇偶位（阴阳属性）
  • 用outer()生成两组向量的所有组合
  • 用paste0()将天干地支拼接为文字
tiangan <- c("甲", "乙", "丙", "丁", "戊", "己", "庚", "辛", "壬", "癸")
dizhi <- c("子", "丑", "寅", "卯", "辰", "巳", "午", "未", "申", "酉", "戌", "亥")

# 提取阳性天干（奇数位）和阴性天干（偶数位）
tg_yang <- tiangan[seq(tiangan) %% 2 == 1]
tg_yin <- tiangan[seq(tiangan) %% 2 == 0]

# 提取阳性地支（奇数位）和阴性地支（偶数位）
dz_yang <- dizhi[seq(dizhi) %% 2 == 1]
dz_yin <- dizhi[seq(dizhi) %% 2 == 0]

# 阳干配阳支
outer(tg_yang, dz_yang, FUN = paste0)
# 阴干配阴支
outer(tg_yin, dz_yin, FUN = paste0)

1.3 矩阵

  • 矩阵：二维，由多个向量组成
  • matrix()创建，按列填充（column-major order）
  • nrow或ncol指定维度（二选一）
  • 矩阵也遵循回收原则
  • 矩阵索引：A[行, 列]
  • 行或列留空代表选择全行或全列
  • 转置：t()，行列互换
  • 两次转置还原

嵌套调用的逻辑

  • “由内而外”：先算最内层，结果作为外层输入
  • 类似数学复合函数 \(g(f(x))\)
  • 矩阵加减法：同维矩阵，对应元素相加减
  • 矩阵乘法：元素积*、内积%*%（前列=后行）、张量积%o%、交叉积crossprod()
  • solve()求解线性方程组
  • 解方程与回归方程的矩阵表示高度相似
